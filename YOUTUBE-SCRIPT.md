# SupAgentic — YouTube Demo Script

## Video Title
**"I Built One CLI to Manage 67 AI Tools — SupAgentic"**

## INTRO (0:00–0:30)
**[Screen: Terminal with SupAgentic banner]**

"If you're working with AI, you probably have a dozen repos scattered everywhere — CrewAI here, Langflow there, Ollama over there. What if I told you there's one CLI that manages all of them?"

**[Type: `supagentic list`]**

"This is SupAgentic — 67 open-source AI tools, 16 categories, 18 commands, one interface."

---

## THE CLI (0:30–2:00)

### List & Search
**[Type: `supagentic list`]**
"Here's every tool organized by category — agents, RAG, coding, math, robotics, voice..."

**[Type: `supagentic search agents`]**
"Search is instant. Find any tool by name, category, or language."

### Clone & Run
**[Type: `supagentic clone langflow`]**
"Clone any tool on-demand — it goes straight to your tools directory."

**[Type: `supagentic setup langflow`]**
"Setup auto-detects requirements.txt, package.json, or pyproject.toml."

**[Type: `supagentic run langflow`]**
"And run auto-detects whether to use npm, python, or docker."

### Stats
**[Type: `supagentic stats`]**
"Live GitHub stats — stars, forks, issues — straight from the API."

### TUI
**[Type: `supagentic tui`]**
"And if you want something interactive, there's a terminal UI browser."

---

## THE MCP SERVER (2:00–3:30)
**[Screen: Claude Desktop]**

"Here's the killer feature. SupAgentic has a full MCP server."

**[Type: `supagentic mcp-register`]**
"One command registers it with Claude Desktop, Cursor, or any MCP client."

**[Switch to Claude Desktop, show tools connected]**
"Now when I talk to Claude, it can see all 67 tools."

**[Type in Claude: "What RAG tools do you have?"]**
"I can ask naturally — 'What RAG tools do you have?' — and it searches my toolkit."

**[Type in Claude: "Clone and set up Haystack"]**
"Or tell it to ACTUALLY do things — 'Clone and set up Haystack' — and it runs clone + setup."

"55 callable tools, 14 resources, 7 prompt templates — all exposed via MCP."
