Attention Residuals
Transformer Architecture
Every LLM has a silent flaw: layer burial. Each layer adds equally to the stack — researchers deleted 40%+ of LLaMA's layers and barely noticed. Early layers get buried and forgotten. Moonshot AI fixed this for nearly free: group layers into chunks → summarize each chunk → every layer attends over those summaries. No need to store all activations. Baked into training, not inference. Result: 48B params, 1.4T tokens, matches a baseline that used 25% MORE compute — better math, coding, and reasoning. Deeper now means smarter, not just bigger.

🔌 MCP Server

Full Model Context Protocol server — any AI agent can discover and use all 171 tools.

🤔 What is MCP?

The Model Context Protocol is an open standard that lets AI assistants (Claude, Cursor, Codex, etc.) connect to external tools and data sources. SupAgentic's MCP server exposes all 171 tools so any AI agent can search, clone, run, setup, and update your tools — just by asking in natural language.

🛠️

171 Tools

Every tool as a callable action — with real execution. Clone, run, setup, update, open from any AI assistant.

📄

15 Resources

README, INTEGRATIONS, CHANGELOG, 7 prompt files, 3 orchestration pipelines, and the GitHub URL Swap Cheatsheet.

💬

7 Prompts

AI prompt templates with argument substitution — chain-of-thought, code review, data analysis, resume builder, and more.

⚡ stdio Transport (recommended)

python mcp_server.py

Direct stdin/stdout — for Claude Desktop, Cursor, Trae, Codex

🌐 SSE/HTTP Transport

python mcp_server.py --sse --port 8765

HTTP/SSE — for remote access, web apps, multi-client setups

🔗 Connect Your AI Client

Auto-register with one command:

python supagentic.py mcp-register

Or add manually to your client's config:

{
  "mcpServers": {
    "supagentic": {
      "command": "python",
      "args": ["path/to/SupAgentic/mcp_server.py"]
    }
  }
}
✅ Claude Desktop ✅ Cursor ✅ Trae ✅ Antigravity / Gemini ✅ Codex Any MCP client

💡 How It Works

1
Register

Run supagentic mcp-register → automatically configures your AI client.

2
Restart Your Client

Restart Claude Desktop, Cursor, or Trae. SupAgentic appears as a connected tool source.

3
Ask Naturally

"What tools do you have for RAG?" → agents see all tools. "Clone and set up Langflow" → it actually runs clone + setup.
